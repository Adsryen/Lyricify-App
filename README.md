### Lyricify Startup Arguments / Lyricify 启动参数
#### Get From / 获取
```
Usage:  
  Lyricify.exe -g getfrom  
  getfrom:  
    0: Spotify (No Login)  
    1: QQ Music  
    2: Netease Cloud Music (or YesPlayMusic)  
    3: Music Center  
    4: iTunes  
Example:  
  Lyricify.exe -g 0
```

#### Font / 字体
```
Usage:  
  Lyricify.exe -font font  
  font:  
    sf: San Francisco  
    segoe: Segoe UI
  There are more for you to discover!  
Example:  
  Lyricify.exe -font sf
```
